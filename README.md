# 🎨 Computer Graphics Projects (OpenGL)

This repository contains a collection of **Computer Graphics projects** implemented using **OpenGL (GLUT)** in C++.

---

## 📂 Projects

---

### 🟢 Drawing Shapes with Mouse Interaction - OpenGlPractices

This project demonstrates:

* Drawing basic shapes
* Mouse interaction
* Rotation using OpenGL

---

### 🎮 Controls

* **Left Click** → Rotate clockwise
* **Right Click** → Rotate counter-clockwise
